# voiceroid mac

 mac用Voiceroidです。

動作には`sayv`コマンドが必要ですが、ライセンスの関係上レポジトリには含めていません。



# 使用方法



![image-20190208102208330](/Users/obuchiyuki/Library/Application Support/typora-user-images/image-20190208102208330.png)

現在以下のVoiceroidに対応しています。

- 結月ゆかり (*)
- 弦巻マキ
- 琴葉 茜・葵  (*)
- 東北ずん子
- 東北きりたん
- 東北イタコ
- 紲星あかり (*)
- 鷹の爪 吉田くん
- 水奈瀬コウ
- 桜乃そら

(*)のあるものでは感情表現設定が可能です。
